Superon

AI 기반 버추얼 캐릭터 자동화 라이브 플랫폼

Superon 이란?

Superon은 AI로 구동되는 가상의 캐릭터가 TikTok, Youtube 등 글로벌 플랫폼에서 팬들과 실시간으로 소통하고 전세계 팬을 얻는 혁신적인 라이브 스트리밍 솔루션입니다. 사용자의 채팅에 실시간으로 반응하며, 감정에 맞는 표정과 목소리로 생동감 넘치는 상호작용을 제공하여 새로운 팬 경험을 만들어냅니다.

핵심 동작 원리

💬

시스템 아키텍처

Superon은 여러 모듈이 유기적으로 연결되어 사용자의 입력을 받아 실시간으로 반응을 생성하고 송출합니다. 아래 흐름도의 각 단계를 클릭하여 해당 모듈의 상세 역할을 확인해보세요.

프로젝트 로드맵

Superon은 지속적인 기술 통합과 기능 확장을 통해 발전하고 있습니다. 현재 주요 과제들의 진행 상황과 앞으로의 확장 계획을 통해 프로젝트의 미래를 확인해보세요.

현재 과제 및 진행상황

  • ✅ LangChain → 감정 분석 → 애니메이션 매핑까지 통합 테스트
  • 🛠️ emotion_tag → motion_tag 매핑 등록 필요 (Live2D)
  • 🔁 OBS websocket 연동 (감정 기반 모션 자동화)
  • 💾 LangSmith memory 기반 응답 지속성 검증 필요

향후 확장 전략

📈 반복 유저 맞춤형 응답
